## üìù Customization

### Update Personal Information

1. **Edit contact details** in `src/components/Footer.jsx`:
   - Update email address
   - Add your social media links (GitHub, LinkedIn, Instagram)
   - Update copyright name

2. **Modify projects** in `src/components/Projects.jsx`:
   - Replace placeholder images with your actual project screenshots
   - Update project titles, descriptions, and tech stacks
   - Add your live demo and GitHub repository links

3. **Update tech stack** in `src/components/About.jsx`:
   - Customize the technologies you use
   - Add or remove tech stack items

4. **Customize content** in `src/components/Hero.jsx`:
   - Update the main headline
   - Modify the typing animation text

### Change Colors

Update the color palette in `tailwind.config.js`:
```javascript
colors: {
  background: '#0A0A12',
  primary: '#8B5CF6',
  secondary: '#A78BFA',
  // ... customize as needed
}
```
